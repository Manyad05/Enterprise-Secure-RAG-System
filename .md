The project uses:

* FastAPI
* Streamlit
* LangChain
* ChromaDB
* HuggingFace Embeddings
* Sentence Transformers

# Installation Guide

# Step 1 - Clone Project

```powershell
git clone <repository-url>
```

OR manually create folders.

---

# Step 2 - Open Project

```powershell
cd enterprise-secure-rag
```

---

# Step 3 - Create Virtual Environment

```powershell
python -m venv venv
```

---

# Step 4 - Activate Virtual Environment

## Windows PowerShell

```powershell
.\venv\Scripts\Activate.ps1
```

Successful activation:

```text
(venv)
```

---

# Step 5 - Fix PowerShell Execution Policy

If activation fails:

```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

Press:

```text
Y
```

Then activate again.

---

# Step 6 - Install Dependencies

```powershell
pip install fastapi uvicorn streamlit requests
```

```powershell
pip install langchain langchain-community
```

```powershell
pip install chromadb
```

```powershell
pip install sentence-transformers
```

```powershell
pip install langchain-openai
```

```powershell
pip install langchain-text-splitters
```

---

# Step 7 - Verify Installation

```powershell
pip show fastapi
```

```powershell
pip show streamlit
```

# Debugging Guide

# Problem 1 - FastAPI Not Found

Error:

```text
ModuleNotFoundError: No module named 'fastapi'
```

Fix:

```powershell
pip install fastapi uvicorn
```

---

# Problem 2 - Streamlit Not Found

Error:

```text
No module named 'streamlit'
```

Fix:

```powershell
pip install streamlit
```

---

# Problem 3 - Virtual Environment Not Activating

Error:

```text
running scripts is disabled
```

Fix:

```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

---

# Problem 4 - Frontend JSON Error

Error:

```text
Expecting value: line 1 column 1
```

Cause:

Backend crashed internally.

Fix:

Check backend terminal logs.

---

# Problem 5 - OpenAI Quota Error

Error:

```text
insufficient_quota
```

Fix:

Use local embeddings:

```python
HuggingFaceEmbeddings
```

---

# Problem 6 - ChromaDB Retrieval Error

Error:

```text
get_relevant_documents
```

Fix:

Replace with:

```python
retriever.invoke(query)
```

---

# Problem 7 - Wrong Python Environment

Check active Python:

```powershell
python -c "import sys; print(sys.executable)"
```

Must point to:

```text
venv/
```

---

# Problem 8 - Missing Chroma Storage

Delete and re-ingest:

```powershell
Remove-Item chroma_storage -Recurse -Force
```

Then:

```powershell
python ingest.py
```
